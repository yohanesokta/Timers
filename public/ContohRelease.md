# Rilis Timer App v0.4.0

Selamat datang di rilis pertama **Timer App**! Aplikasi timer minimalis dan modern yang dirancang untuk fokus dan produktivitas Anda.

## Fitur Utama

- **Tampilan Waktu Presisi Tinggi**: Timer dengan format Jam, Menit, Detik, hingga milidetik (`HH:MM:SS:LL`) untuk akurasi maksimal.
- **Desain Modern & Responsif**: Tampilan yang bersih dan dapat menyesuaikan diri dengan berbagai ukuran layar, dari desktop hingga perangkat mobile.
- **Tema Terang & Gelap**: Ganti antara mode terang dan gelap untuk kenyamanan visual Anda kapan saja.
- **Mode Layar Penuh**: Masuk ke mode layar penuh untuk pengalaman yang imersif dan bebas gangguan.
- **Progress Bar Dinamis**: Sebuah bar progres visual yang menunjukkan sisa waktu, berubah warna dari hijau menjadi kuning, lalu merah saat waktu hampir habis.
- **Peringatan 3 Menit**: Tampilan timer akan berubah menjadi merah dan berdenyut ketika waktu tersisa kurang dari 3 menit untuk memberikan peringatan visual.
- **Watermark**: Terdapat watermark elegan dengan logo GitHub di pojok kiri atas.

### Cara Verifikasi Checksum (Opsional)

Untuk memastikan integritas file yang Anda unduh, Anda dapat memverifikasi checksum SHA-256.

- **Di Windows (PowerShell):**
  ```powershell
  Get-FileHash .\[nama-file] -Algorithm SHA256
  ```

- **Di Linux:**
  ```sh
  sha256sum [nama-file]
  ```
